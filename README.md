## Документація про можливості API:
-  ### User модуль:
    1. ##### `/user/login` **Авторизація** (`POST`): Авторизація користувача за його даними.
        - В розробці
    2. ##### `/user/register` **Реєстрація** (`POST`): Створення нового акаунта користувача за його даними.
        - В розробці
    3. ##### `/user/generate-code` **Генерація коду** (`POST`): Генерація коду підтвердження пошти, який годний 15 хв.
        - В розробці
    4. ##### `/user/validate-code` **Перевірка коду** (`POST`): Перевіряє код, надісланий користувачем, на співпадання у базі та його час.
        - В розробці
    6. ##### `/user/me` **Реєстрація** (`GET`): Отримання власних даних авторизованого користувача.
        - В розробці
